\begin{DoxyItemize}
\item First we test the parameter integrity as a precaution. If we have bad parameters we return immediately. 


\item We return immediately if the N/\-P\-C is in Combat \char`\"{}\-Compete\char`\"{} Mode. \begin{DoxySeeAlso}{See Also}
\hyperlink{fight_8cpp_a36f0075afb877880bc0476049a872353}{do\-\_\-compete()} 
\end{DoxySeeAlso}

\item Examine {\ttfamily type} to determine who receives this action message. 


\item Iterate through persons in the recipient's room sending the action message where appropriate. 
\begin{DoxyItemize}
\item We do not translate and send the message\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} the recipient is not the actor  
\item {\itshape A\-N\-D} the actor is immortal  
\item {\itshape A\-N\-D} the recipient is mortal  
\item {\itshape A\-N\-D} the actor is invisible. 


\end{DoxyItemize}


\item We translate and send the message\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} the recipient is connected (i.\-e. a link-\/live P\-C or an animated N\-P\-C)  
\item {\itshape A\-N\-D} the recipient is not the actor ({\itshape O\-R} this message type is to the actor)  
\item {\itshape A\-N\-D} the recipient is in the actor's group ({\itshape O\-R} this message type is not group-\/only)  
\item {\itshape A\-N\-D} the recipient can see the actor ({\itshape O\-R} we do not want to hide invisible actors  
\item {\itshape A\-N\-D} the recipient is awake  
\item {\itshape A\-N\-D} the type is not immortal-\/only ({\itshape O\-R} the recipient is on staff and the actor is not (?))  
\item {\itshape A\-N\-D} {\itshape N\-O\-T} a non-\/victim message {\itshape I\-F} the recipient is the same as the victim.  
\end{DoxyItemize}


\item Iterate through the action\-\_\-message and output buffer character-\/at-\/a-\/time. 
\begin{DoxyItemize}
\item We parse the next character as a format identifier {\itshape I\-F} this character is {\ttfamily \$}. {\itshape E\-L\-S\-E} we copy this character from the message to the buffer (stopping when we reach the end of the string. 


\item Format translations\-: 
\begin{DoxyItemize}
\item {\ttfamily \$n} -\/ insert the short description of actor (showing \char`\"{}someone\char`\"{} if the recipient cannot see, and storing the actor's name for immortals. 


\item {\ttfamily \$\-N} -\/ insert the short description of the victim (showing \char`\"{}someone\char`\"{} if the recipient cannot see, and storing the victim's name for immortals to see. 


\item {\ttfamily \$3} -\/ insert the short description of third P\-C (?). 


\item {\ttfamily \$m} -\/ actor as \char`\"{}him\char`\"{}, \char`\"{}her\char`\"{}, or \char`\"{}it\char`\"{} (the latter in the case of neuter or hooded actors). 


\item {\ttfamily \$\-M} -\/ victim as \char`\"{}him\char`\"{}, \char`\"{}her\char`\"{}, or \char`\"{}it\char`\"{} (the latter in the case of neuter or hooded victims). 


\item {\ttfamily \$s} -\/ actor as \char`\"{}his\char`\"{}, \char`\"{}her\char`\"{}, or \char`\"{}its\char`\"{} (the latter in the case of neuter or hooded actors). 


\item {\ttfamily \$\-S} -\/ victim as \char`\"{}his\char`\"{}, \char`\"{}her\char`\"{}, or \char`\"{}its\char`\"{} (the latter in the case of neuter or hooded victims). 


\item {\ttfamily \$e} -\/ actor as \char`\"{}he\char`\"{}, \char`\"{}she\char`\"{}, or \char`\"{}it\char`\"{} (the latter in the case of neuter or hooded actors). 


\item {\ttfamily \$\-E} -\/ victim as \char`\"{}he\char`\"{}, \char`\"{}she\char`\"{}, or \char`\"{}it\char`\"{} (the latter in the case of neuter or hooded victims). 


\item {\ttfamily \$o} -\/ object name or \char`\"{}something\char`\"{} if hidden from the recipient 


\item {\ttfamily \$\-O} -\/ victim object name or \char`\"{}something\char`\"{} if hidden from the recipient 


\item {\ttfamily \$p} -\/ object short description or \char`\"{}something\char`\"{} if hidden from the recipient 


\item {\ttfamily \$\-P} -\/ victim object short description or \char`\"{}something\char`\"{} if hidden from the recipient 


\item {\ttfamily \$a} -\/ \char`\"{}a\char`\"{} or \char`\"{}an\char`\"{} object (based on name) 


\item {\ttfamily \$\-A} -\/ \char`\"{}a\char`\"{} or \char`\"{}an\char`\"{} victim object (based on name) 


\item {\ttfamily \$\-T} -\/ Cast parameter {\ttfamily vict\-\_\-obj} as a string and set to local {\ttfamily i} (?). 


\item {\ttfamily \$\-F} -\/ Cast parameter {\ttfamily vict\-\_\-obj} as a string of names and set to local {\ttfamily i} to the first of them (?). 


\item {\ttfamily \$g} -\/ insert the name of the actor's god (?) 


\item {\ttfamily \$\-G} -\/ insert the name of the victim's god (?). 


\item {\ttfamily \$\$} -\/ Insert a dollar-\/sign 


\end{DoxyItemize}


\item Do not copy local {\ttfamily i} to {\ttfamily buf} {\itshape I\-F} there is nothing to insert ({\itshape O\-R} we are at the end of the buffer), {\itshape E\-L\-S\-E} Capitalize, color, and copy.


\end{DoxyItemize}


\item Cap the end of the output buffer with a newline \& null. 


\item We do not send the buffer\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} this is a combat message  
\item {\itshape A\-N\-D} the recipient filters combat messages  
\item {\itshape A\-N\-D} the recipient is not fighting the victim 
\item {\itshape A\-N\-D} the recipient is not fighting the actor 
\item {\itshape A\-N\-D} the victim is not fighting the recipient 
\item {\itshape A\-N\-D} the actor is not fighting the recipient

\item We do not send the buffer\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} this is a firefight message  
\item {\itshape A\-N\-D} the recipient filters firefight messages  
\item {\itshape A\-N\-D} the recipient is not fighting the victim 
\item {\itshape A\-N\-D} the recipient is not fighting the actor 
\item {\itshape A\-N\-D} the victim is not fighting the recipient 
\item {\itshape A\-N\-D} the actor is not fighting the recipient 
\end{DoxyItemize}


\item We do not send the buffer\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} this is a bleeding message  
\item {\itshape A\-N\-D} the recipient filters combat messages  
\item {\itshape A\-N\-D} the recipient is not fighting the victim 
\item {\itshape A\-N\-D} the recipient is not fighting the actor 
\item {\itshape A\-N\-D} the victim is not fighting the recipient 
\item {\itshape A\-N\-D} the actor is not fighting the recipient 
\item {\itshape A\-N\-D} the recipient is fighting 
\end{DoxyItemize}


\item We do not send the buffer\-: 
\begin{DoxyItemize}
\item {\itshape I\-F} this is a search/scan message  
\item {\itshape A\-N\-D} the recipient filters combat messages  
\item {\itshape A\-N\-D} the recipient is grouped or admin/li$>$ 
\end{DoxyItemize}


\item Reformat the buffer {\itshape I\-F} the {\ttfamily type} parameter is set accordingly  
\item Send the buffer to the recipient 


\item {\itshape I\-F} the action {\ttfamily type} is to a single recipient, we can stop iterating. {\itshape E\-L\-S\-E} process the next recipient.


\end{DoxyItemize}
\item Return to the caller after the message has been sent to any recipients  
\end{DoxyItemize}
